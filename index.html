What is CyNetShare?

CyNetShare has been depricated. Use the Network Data Exchange (NDEx) project for network storing and exchanging.

CyNetShare is an easy way to share your network data files created with Cytoscape.

Easy to Use
Users can share Cytoscape.js interactive visualizations without programming.
Simple
Once file is available as a public URL, it can be shared by copy-paste it
Support Desktops and Tablets
It is a pure HTML5 application and supports both desktops and tablets.
Open Source
CyNetShare is an open source project. Developers can use it as a boilarplate code for their web applications.

Background

  • What is Cytoscape?

    Cytoscape is an open source platform for network analysis and visualizarion. You can prepare your network data files for CyNetShare with Cytoscape and its variety of Apps.

  • Cytoscape Cyberinfrastructure

    Cytoscape was developed as a desktop application, but now its ecosystem is expanding. CyNetShare is a pilot project towards the Cytoscape Cyberinfrastructure, an open network analysis environment.
